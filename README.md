# 🌱 ECO-STEP | Sustainable Footwear Website

ECO-STEP is a modern, responsive landing page focused on sustainable footwear made from recycled ocean plastic.  
The project highlights environmental awareness, product storytelling, and interactive user experience.

## 🌍 About the project

This website presents a concept of eco-friendly sneakers created from marine plastic waste.  
It combines clean design, smooth animations, and interactive elements to deliver a premium digital experience.

The main goal is to:
- promote sustainability and conscious consumption
- demonstrate how recycled materials can be transformed into high-quality products
- create an engaging and modern UI/UX experience

---

## 🚀 Technologies used

- HTML5 (semantic structure)
- CSS3 (custom design system, animations, responsive layout)
- JavaScript (DOM manipulation, animations, interactivity)

---

## ⚙️ Features

### 🎯 Interactive UI
- Dynamic hero text animation (changing keywords: *Heals / Protects / Restores*)
- Smooth scrolling navigation between sections
- Interactive buttons with scroll behavior

### 🖼 Product Experience
- Hover zoom effect on product images
- Dynamic transform based on cursor position
- Clean product card layout

### 🎬 Animations
- Scroll-based reveal animations using Intersection Observer
- Animated counters (e.g. bottles recycled, waste reduced)
- Micro-interactions and hover effects

### 📩 Contact System
- Modal window with contact form
- Open/close interactions (click, outside click, ESC key)
- Basic form submission handling

### 📱 Responsive Design
- Fully adaptive layout for desktop, tablet, and mobile devices

---

## 🎨 Design Highlights

- Minimalistic eco-inspired color palette
- Glassmorphism effects (blurred navigation, cards)
- Soft gradients and background textures
- Strong typography using Google Fonts (Plus Jakarta Sans)
